# SKYJOS - NOTAM Analyzer & Visualizer

An aeronautical tool designed to parse complex NOTAM (Notice to Air Missions) data and visualize them on an interactive map.

## Features
- **Intelligent Parsing**: Extracts coordinates, altitudes, dates, and aeronautical fixes (VOR/VORTAC/INT) from raw NOTAM text.
- **Interactive Map**: Renders Area (Polygons), Route (Lines), and Radius (Circles) geometries.
- **Overlapping NOTAM Navigation**: Cycle through multiple NOTAMs in a single location with a built-in navigator.
- **Advanced Filtering**: Filter by minimum altitude and date range.
- **Visual Feedback**: Real-time selection highlighting in orange.
- **Data Export**: Copy details as JSON or download as CSV.

## Getting Started
Simply open `public/index.html` in your web browser.
